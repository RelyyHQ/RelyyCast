# Contributing to RelyyCast

Thank you for your interest in contributing to RelyyCast!

Contribution guidelines are currently being written and will be published here soon. In the meantime, please feel free to:

-   **Open an issue** to report a bug or propose a feature. Check existing issues first to avoid duplicates.
-   **Submit a pull request** against the `main` branch for small, clearly scoped fixes.
-   **Review the [Developer Setup](README.md#developer-setup)** section of the README to get your local environment running.

For questions or broader discussions, visit [relyycast.com](https://relyycast.com).
